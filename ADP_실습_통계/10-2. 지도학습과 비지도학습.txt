10-2. 지도학습과 비지도학습
 - 머신러닝은 데이터의 훈련 샘플들을 학습해 데이터의 규칙과 패턴을 학습하여 결과를 예측하는 알고리즘과 시스템을 연구하는 분야
 . 지도학습 : Target 예측을 위해 데이터를 학습시킴
 . 비지도 : Target이 없이 데이터의 특성을 추출
 - 통계학에서 유래된 머신러닝 기법들이 많음
 - 머신러닝에서는 데이터에 대한 규칙을 프로그래밍으로 구현하지 않아도 파이썬 API 기반의 라이브러리로 손쉽게 모델을 구축할 수 있음

ㅁ 지도학습 : 레이블된 데이터를 학습하여 미래 데이터 예측
ㅁ 비지도학습
 - 레이블 없이 학습. 
 - 데이터의 숨겨진 특징 발견
 - PCA, 군집분석  

ㅁ 지도&비지도 학습
 - 분류분석 : 학습된 데이터로 종속변수의 Class 예측
 - 회귀분석 : 종속변수의 값을 예측 / 종속변수와 독립변수의 관계 정량화
 - 군집분석 : 데이터의 특성에 따라 묶음으로 유형별 특징을 분석
 - 차원축소 : 다차원이 데이터 셋을 축소하여 새로운 차원의 데이터 셋으로 변환
